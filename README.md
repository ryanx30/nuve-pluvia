# NUVE PLUVIA — Waterproofing Specialist Website

Official company profile website for **NUVE Pluvia Solution**, a specialist waterproofing provider focused on long-term building protection.

Built with **Next.js (App Router)** and **Tailwind CSS**.

---

## Live Pages

- **Landing Page**: `/`
- **Home**: `/site/home`
- **Product**: `/site/product`
- **About Us**: `/site/about-us`

The **“Get a Solution”** button on the landing page redirects users to **Home** (`/site/home`).

---

## Features

- Responsive landing page (mobile & desktop)
- Dedicated site section with consistent navigation
- Optimized images via `next/image`
- Clean UI system with Tailwind CSS
- Separate layouts:
  - Landing page without Navbar/Footer
  - Main site pages with Navbar/Footer

---

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
